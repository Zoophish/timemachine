## Time Machine

Various components for time series forecasting projects.

### Outline   
* Models   
    * Stateful LSTM   
    * CNNLSTM   
    * DFLSTM   
* Data   
    * Artificial time series data   
    * Financial data   
    * Meteorological data   
* Preprocessing   
    * Kalman filters   
    * Cyclical calendar encodings   
    * Rolling features   
* Training   
    * Early stopper   
    * Loss functions

### Requirements

* PyTorch   
* Polars   
* Pandas   
* Numpy   
* Joblib
